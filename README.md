# ☕ Starbucks Power BI Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes Starbucks beverage data to provide insights into calories, caffeine, sugar, beverage categories, and global presence.

## 📊 Dashboard Features
- Total Beverage Count
- Average Calories
- Average Sugar
- Average Caffeine
- Beverage Category Distribution
- Average Calories by Category
- Average Caffeine by Category
- Top 5 Highest Caffeine Beverages
- Starbucks Global Presence Map
- Interactive Slicers (Protein Range & Beverage Prep)

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Bing Maps
